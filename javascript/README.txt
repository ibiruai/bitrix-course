РАБОТА ПО МОДУЛЮ ОСНОВЫ ПРОГРАММИРОВАНИЯ

Задание:
https://github.com/netology-code/pb-diplom/

После знакомства с JavaScript нужно было создать игру Крестики-нолики.
Внешний вид игры уже был готов, задачей было написать её логику.

В распоряжении были следующие файлы:

- logic.js — пустой файл, в котором нужно было написать код игры.
- index.html, styles.css, ui.js — готовые файлы, их менять не нужно.

В logic.js нужно было реализовать две функции:

1. startGame
Вызывается при запуске или перезапуске игры.
В этой функции нужно:
- создать игровое поле,
- выбрать, кто из игроков ходит первым.

2. click
Вызывается при клике по клетке поля.
Получает два параметра: номер строки и номер колонки.
В этой функции нужно:
- обновить поле, поставив крестик или нолик,
- проверить, выиграл ли игрок,
- если никто не выиграл — передать ход другому игроку.
